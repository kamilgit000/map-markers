
## Map-markers

App that alows you to add, edit and delete pins on the map.<br>

[demo](https://map-markers.herokuapp.com/)

## How to run
In the project directory, you can run:

### `npm i`
Then after dependencies are installed, run:

### `npm start`
Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.\
You will also see any lint errors in the console.



## Used technologies
```
[framework]: react
[state management]: redux toolkit & redux-persist
[UI]: mui & styled components
[language]: typescript
[routing]: react router dom v6
[map]: pigeon-maps
[deployment]: heroku
```

## Deployment

App deployment is automated and its hooked directly to new commits on branch `main`
